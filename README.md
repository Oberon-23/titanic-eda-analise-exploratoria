## Sobre o Projeto

Este projeto apresenta uma **Análise Exploratória de Dados (EDA)** completa sobre o famoso dataset do Titanic, disponível na plataforma Kaggle.

O naufrágio do RMS Titanic em 1912 é um dos acidentes marítimos mais conhecidos da história. A partir dos dados dos passageiros, investigamos:

- 🔍 Quem eram os passageiros do Titanic?
- 📊 Quais fatores influenciaram a sobrevivência?
- 💡 O que os dados revelam sobre desigualdade social e prioridade no resgate?

## Principais Insights

- **Sexo foi o fator mais determinante** — mulheres tiveram ~74% de sobrevivência vs ~19% dos homens
- **Classe social impactou diretamente** — 1ª classe teve ~63% de sobrevivência vs ~24% na 3ª classe
- **Crianças tiveram prioridade** — passageiros mais jovens apresentaram maior taxa de sobrevivência
- **Ter cabine aumentou as chances** — passageiros com cabine eram majoritariamente da 1ª e 2ª classe
- **"Mulheres e crianças primeiro"** — os dados confirmam essa prioridade no resgate
